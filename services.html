Portfolio

Présentation de 3 projets qui me tiennent à coeur dont j'ai réalisés pendant mes 3 années de bac !

GeoRisk

GeoRisk est une application réalisée avec ElectronJS pour visualiser les risques géologiques et météorologiques. Les informations sont basées sur les données du gouvernement français disponibles sur leur site internet.

GestMoney

GestMoney est une application réalisée en Python avec le module Tkinter. Elle nous permet de gérer nos comptes bancaires et de les suivre. Bien sûr, aucune donnée bancaire n'est requise

SpaceX

SpaceX est un logiciel fonctionnant dans dans le terminal qui, en fonction des étapes choisies, permet de récupérer des données dans l'espace. C'est un logiciel pour les passionnés de l'astronomie. Il est actuellement référencé sur plusieurs sites tels que : PythonAwesome
